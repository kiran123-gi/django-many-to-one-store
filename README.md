🛒 Shopping Store - Many-to-One Relationship

This project is a Django-based backend application demonstrating a Many-to-One relationship between Customers and Products.

🚀 Features:

* Create Customers
* Assign multiple Products to a Customer
* View products based on customer
* Delete Customer with cascade delete (removes related products automatically)

🛠️ Tech Stack:

* Python
* Django
* SQLite

📌 Core Concept:

* Many-to-One Relationship using ForeignKey
* CASCADE delete behavior in Django ORM

📂 Use Case:
This project helps beginners understand relational database design and how parent-child relationships work in real-world applications.
